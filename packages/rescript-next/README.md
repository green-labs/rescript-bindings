# `rescript-next`

ReScript binding of [Next.js](https://nextjs.org)

## Install

```bash
npm i @greenlabs/rescript-next
or
yarn add @greenlabs/rescript-next
```

```json
"bs-dependencies": [
  "@greenlabs/rescript-next"
]
```

## Usage

```rescript
<Next.Head>
  <title> {`rescript-next`->React.string} </title>
</Next.Head>
```
