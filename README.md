## Development

**Install dependencies**

```shell
yarn
```

**Run Build**

Must be executed before open PR.

```shell
yarn build
```

**Add new package**

```shell
yarn add-package -n "rescript-${package-name}"
```

## Release

- use [changesets](https://github.com/changesets/changesets)

```shell
yarn changeset add
```
